# Notes Taking App 
This is a simple notes app built with Frontend in React with Django and Mysql.

## Requirements
1. Python 3.9
2. Node.js
3. React

## Installation
1. Clone the repository

2. Build the app
```
docker build -t notes-app .
```

3. Run the app
```
docker run -d -p 8000:8000 notes-app:latest
```
